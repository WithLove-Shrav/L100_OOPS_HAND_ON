# Constructors & Destructors in TypeScript

This folder demonstrates how **constructors** are used to initialize class instances in TypeScript. While Java has explicit destructors, TypeScript (and JavaScript) relies on garbage collection, so there is no direct destructor concept.

- Classes with parameterless and parameterized constructors.

- A brief note on cleanup patterns (e.g., `dispose` methods) for resources.

## How to explore
1. Open the `.ts` files and examine the constructor definitions.
2. Compile with `tsc` or run directly with `ts-node`.

```bash
# Run directly (requires ts-node)
npx ts-node example.ts
# Or compile then run
npx tsc example.ts
node example.js
```

Feel free to add your own constructors or experiment with default values.
